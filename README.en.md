# AI-EdVision - 智能教育视频分析与个性化辅导系统

#### Description
- 用户通过平台上传或提供视频链接，系统自动处理视频并生成实时字幕
- 在观看视频时，用户可以：
  1. 实时记录带时间戳的笔记
  2. 提出视频相关或无关的问题，获得智能回答
  3. 获取个性化的学习路径建议
- 系统持续分析用户笔记，推荐相关内容，优化学习体验
