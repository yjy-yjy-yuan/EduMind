# AI-EdVision 智能教育视频分析系统 - 可执行文档

本文档详细说明了AI-EdVision系统的功能模块及其执行方法，帮助用户快速上手并使用系统的各项功能。

## 系统概述

AI-EdVision是一个智能教育视频分析系统，旨在通过AI技术提升教育视频的学习效果。系统能够自动处理教育视频，提取关键知识点，构建知识图谱，并提供个性化学习路径推荐。

## 环境要求

### 硬件要求
- CPU: 推荐Intel Core i5/i7或AMD Ryzen 5/7以上处理器
- 内存: 至少8GB RAM，推荐16GB或更高
- 存储空间: 至少20GB可用空间
- GPU: 推荐NVIDIA GPU（用于加速视频处理和AI模型）

### 软件要求
- 操作系统: Windows 10/11 64位
- Python 3.10
- Node.js 16+
- Neo4j数据库
- Redis内存数据库
- FFmpeg视频处理工具
- Ollama本地大语言模型部署工具

## 系统安装与部署

### 1. 环境配置
1. **安装Python 3.10**
   - 访问[Python官网](https://www.python.org/downloads/release/python-3100/)下载Python 3.10
   - 安装时勾选"Add Python to PATH"选项
   - 验证安装：打开命令提示符，输入`python --version`确认版本为3.10.x

2. **安装Node.js**
   - 访问[Node.js官网](https://nodejs.org/)下载Node.js 16+版本
   - 按照默认选项完成安装
   - 验证安装：打开命令提示符，输入`node --version`和`npm --version`确认版本

3. **安装Redis**
   - 访问[Redis for Windows](https://github.com/microsoftarchive/redis/releases)下载最新的Redis-x64-xxx.msi安装包
   - 双击安装包进行安装
   - 安装时选择"Add the Redis installation folder to the PATH environment variable"
   - 验证安装：打开命令提示符，输入`redis-cli --version`确认安装成功

4. **安装FFmpeg**
   - 访问[FFmpeg官网](https://ffmpeg.org/download.html)下载FFmpeg
   - 解压下载的文件到指定目录（如C:\ffmpeg）
   - 将FFmpeg的bin目录添加到系统环境变量PATH中
   - 验证安装：打开命令提示符，输入`ffmpeg -version`确认安装成功

5. **安装Neo4j**
   - 访问[Neo4j官网](https://neo4j.com/download/)下载Neo4j Desktop
   - 按照安装向导完成安装
   - 创建一个新的数据库，设置用户名和密码（默认用户名为neo4j）
   - 启动数据库并记录连接信息（URI、用户名和密码）

6. **安装Ollama并下载模型**
   - 访问[Ollama官网](https://ollama.com/)下载并安装Ollama
   - 打开命令提示符，运行以下命令下载所需模型：
     ```bash
     ollama pull deepseek-r1:8b
     ollama pull qwen2.5:7b
     ```

### 2. 项目部署
1. **克隆项目**
   ```bash
   git clone https://github.com/yourusername/AI-EdVision.git
   cd AI-EdVision
   ```

2. **配置数据库连接**
   - 修改Neo4j数据库连接信息：
     - 打开`backend/app/utils/knowledge_graph_utils.py`文件
     - 找到`KnowledgeGraphManager`类的`__init__`方法
     - 修改数据库连接信息为您自己的配置：
     ```python
     def __init__(self, uri="bolt://localhost:7687", user="neo4j", password="your_password", similarity_service=None):
     ```
   - 同样修改`backend/app/routes/knowledge_graph.py`文件中的数据库连接信息

3. **配置环境变量**
   - 在`backend`目录下创建`.env`文件（如果不存在）
   - 添加以下配置：
     ```
     FLASK_APP=run.py
     FLASK_ENV=development
     SECRET_KEY=your_secret_key
     DATABASE_URL=sqlite:///app.db
     ```

4. **安装后端依赖**
   ```bash
   # 创建并激活虚拟环境
   cd backend
   python -m venv venv
   venv\Scripts\activate  # Windows
   # 或使用Conda
   conda create -n ai-edvision python=3.10 -y
   conda activate ai-edvision
   
   # 安装基本依赖
   pip install -r requirements.txt
   
   # 安装PyTorch（根据您的环境选择）
   # GPU环境（CUDA 11.8）
   conda install pytorch==2.0.0 torchvision==0.15.0 torchaudio==2.0.0 pytorch-cuda=11.8 -c pytorch -c nvidia
   # 或CPU环境
   pip install torch==2.0.0 torchvision==0.15.0 torchaudio==2.0.0 --index-url https://download.pytorch.org/whl/cpu
   
   # 安装FAISS（根据您的环境选择）
   # GPU环境
   conda install -c conda-forge faiss-gpu=1.8.0 -y
   # 或CPU环境
   conda install -c conda-forge faiss-cpu=1.8.0 -y
   
   # 初始化数据库
   python create_tables.py
   ```

5. **安装前端依赖**
   ```bash
   cd ../frontend
   npm install
   ```

## 系统启动与运行

### 1. 启动系统
1. **启动Redis服务**
   ```bash
   # 在命令提示符或PowerShell中运行
   redis-server
   
   # 在另一个终端中验证Redis是否正常运行
   redis-cli ping  # 应返回PONG
   ```

2. **启动Celery Worker（处理异步任务）**
   ```bash
   # 在backend目录下
   cd backend
   venv\Scripts\activate  # 或 conda activate ai-edvision
   python -m celery -A app.celery_app worker --loglevel=info -P solo
   ```

3. **启动Flask后端服务**
   ```bash
   # 在另一个终端，backend目录下
   venv\Scripts\activate  # 或 conda activate ai-edvision
   python run.py
   ```

4. **启动前端开发服务器**
   ```bash
   # 在另一个终端，frontend目录下
   npm run dev
   # 或允许局域网访问
   npm run dev --host
   ```

5. **在浏览器中访问系统**
   - 本地访问：http://localhost:5173
   - 局域网访问：http://[您的IP地址]:5173 (使用`--host`选项时)

### 2. 系统功能使用指南

#### 2.1 视频上传与处理
1. 点击首页的"开始上传视频"按钮或导航栏的"上传视频"选项
2. 选择本地视频文件上传或输入视频链接（支持哔哩哔哩/YouTube）
3. 系统会自动处理视频，包括视频转码、字幕提取、关键帧分析等
4. 处理完成后，可以在视频列表中查看和管理已上传的视频

#### 2.2 视频播放与学习
1. 在视频列表中点击视频缩略图或标题进入视频详情页
2. 点击"开始学习"按钮进入视频播放页面
3. 视频播放页面提供以下功能：
   - 智能字幕显示（支持分段和语义合并显示）
   - 视频内容问答（可随时提问视频相关问题）
   - 笔记记录（可在观看视频时记录笔记）
   - 知识点标记（系统自动标记视频中的关键知识点）

#### 2.3 知识图谱浏览
1. 点击导航栏的"知识点总览"选项进入知识图谱页面
2. 知识图谱页面展示了所有视频中提取的知识点及其关联关系
3. 可以通过以下方式与知识图谱交互：
   - 缩放和平移图谱
   - 点击知识点查看详细信息
   - 搜索特定知识点
   - 按类别筛选知识点
   - 查看知识点相关的视频和笔记

#### 2.4 智能问答系统
1. 在视频播放页面或知识图谱页面可以使用智能问答功能
2. 输入问题后，系统会基于视频内容和知识库提供回答
3. 支持以下问答模式：
   - 快速回答模式：直接基于已有知识回答
   - 深度思考模式：结合多个知识点进行推理后回答
   - 在线/离线模式：可选择使用在线API或本地模型

#### 2.5 笔记管理系统
1. 点击导航栏的"笔记系统"选项进入笔记管理页面
2. 笔记管理页面提供以下功能：
   - 查看所有笔记
   - 按视频、日期、标签等筛选笔记
   - 编辑和删除笔记
   - 导出笔记（支持Markdown、PDF等格式）
   - 查看相似笔记推荐

#### 2.6 个人资料管理
1. 点击导航栏右上角的用户头像，选择"个人资料"选项
2. 个人资料页面可以：
   - 查看和编辑个人信息
   - 设置学习偏好
   - 查看学习统计数据
   - 管理账户设置

## 常见问题与解决方案

### 1. 系统启动问题
- **问题**：Redis服务无法启动
  **解决方案**：检查Redis是否正确安装，尝试以管理员身份运行命令提示符启动Redis服务

- **问题**：后端服务启动失败
  **解决方案**：检查环境变量配置，确保所有依赖已正确安装，查看错误日志获取详细信息

- **问题**：前端开发服务器启动失败
  **解决方案**：检查Node.js版本，尝试删除node_modules文件夹并重新安装依赖

### 2. 视频处理问题
- **问题**：视频上传后处理失败
  **解决方案**：检查视频格式是否支持，确保FFmpeg正确安装，查看后端日志获取详细错误信息

- **问题**：字幕提取不准确
  **解决方案**：对于噪音较大或语音不清晰的视频，可以尝试手动上传字幕文件

### 3. 知识图谱问题
- **问题**：知识图谱无法显示
  **解决方案**：检查Neo4j数据库连接配置，确保数据库服务正在运行

- **问题**：知识点提取不准确
  **解决方案**：可以通过知识图谱页面的编辑功能手动调整知识点

### 4. 其他常见问题
- **问题**：系统响应缓慢
  **解决方案**：检查硬件资源使用情况，考虑增加内存或使用GPU加速

- **问题**：无法登录系统
  **解决方案**：检查用户名和密码，如忘记密码可使用"忘记密码"功能重置
