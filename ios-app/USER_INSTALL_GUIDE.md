# EduMind iOS 安装指南（免费版）

## 准备工作

1. 一台 Mac 电脑
2. 一条 iPhone 数据线
3. iPhone 手机（iOS 12.0+）
4. 免费的个人 Apple ID（已有即可）

## 安装步骤

### 第一步：在 Mac 上安装 AltServer

1. 访问 https://altstore.io/
2. 点击「Download AltServer」下载
3. 将下载的文件复制到「应用程序」文件夹
4. 双击打开 AltServer
5. 系统会提示安装 Mail 插件，选择「安装」

### 第二步：在 iPhone 上安装 AltStore

1. 在 iPhone 上打开 Safari，访问 https://altstore.io/download/
2. 点击蓝色按钮「Download AltStore」
3. 系统会跳转到设置页面，点击「安装」
4. 安装完成后，在主屏会出现 AltStore 图标

### 第三步：连接设备并信任

1. 用数据线将 iPhone 连接到 Mac
2. 在 Mac 上，点击顶部菜单栏的 AltServer 图标
3. 选择「Install AltStore」→ 选择你的 iPhone
4. 输入你的 Apple ID 和密码（免费账号即可）
5. 在 iPhone 上，前往「设置」→「通用」→「VPN 与设备管理」
6. 找到你的 Apple ID 开发者证书，点击「信任」

### 第四步：安装 EduMind

1. 从开发者处获取 `EduMindIOS.ipa` 文件
2. 将 IPA 文件保存到 Mac 的某个位置（如桌面）
3. 确保 iPhone 和 Mac 在同一 Wi-Fi 网络
4. 在 Mac 上，点击顶部菜单栏的 AltServer 图标
5. 双击 `EduMindIOS.ipa` 文件
6. 在 iPhone 上，点击「安装」
7. 等待安装完成

### 第五步：首次启动

1. 在 iPhone 主屏找到 EduMind 图标
2. 点击启动应用
3. 如果提示「未受信任的开发者」，前往设置信任（同第三步）
4. 开始使用 EduMind！

## 定期续签（每 7 天）

为了保持应用可正常使用，每 7 天需要续签一次：

1. 确保 iPhone 和 Mac 在同一 Wi-Fi 网络
2. 在 Mac 上运行 AltServer（如果没运行）
3. 在 iPhone 上打开 AltStore
4. 找到 EduMind，点击「Refresh」
5. 输入 Apple ID 密码
6. 等待续签完成

## 常见问题

### 安装失败？
- 检查 Mac 和 iPhone 是否在同一 Wi-Fi
- 确保 AltServer 正在运行（顶部菜单栏有图标）
- 重新连接数据线
- 重新信任开发者证书

### 7 天后打不开应用？
- 按照上述「定期续签」步骤操作
- 续签后即可正常使用

### 可以安装到其他设备吗？
- 可以，但每台设备都需要独立的 AltStore 安装
- 单个 Apple ID 最多支持 3 台设备

## 需要帮助？

- 联系 EduMind 开发者

- 建议将 AltServer 设置为开机自启动，方便后续续签
- 记得定期更新 EduMind 到最新版本
- 应用数据会保存在本地，更新不会丢失
