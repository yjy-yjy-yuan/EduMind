# Sideloadly 安装指南

## 什么是 Sideloadly？

Sideloadly 是一个免费的 iOS 应用侧载工具，支持 Windows 和 Mac，可以无需付费苹果开发者账号就安装 IPA 文件。

## 系统要求

- **Windows**: Windows 7 或更高版本
- **Mac**: macOS 10.12 或更高版本
- **iOS**: iOS 12.0 或更高版本
- **数据线**: 需要用数据线连接 iPhone 和电脑

## 安装步骤

### Windows 用户

#### 1. 下载 Sideloadly
1. 访问 https://sideloadly.io/
2. 下载 Windows 版本（建议下载最新版）
3. 安装程序并运行

#### 2. 安装 iTunes（如果还没有）
- 下载并安装 iTunes（Sideloadly 依赖 iTunes）
- 确保可以正常连接 iPhone

#### 3. 安装应用
1. 用数据线连接 iPhone 和电脑
2. 打开 Sideloadly
3. 点击「Browse」选择 `EduMindIOS.ipa` 文件
4. 在「Device」下拉菜单选择你的 iPhone
5. 在「Apple ID」栏输入你的 Apple ID 和密码
6. 点击「Start」开始安装
7. 在 iPhone 上信任开发者证书：
   - 设置 → 通用 → VPN 与设备管理 → 信任

### Mac 用户

#### 1. 下载 Sideloadly
1. 访问 https://sideloadly.io/
2. 下载 macOS 版本
3. 将下载的文件移到应用程序文件夹

#### 2. 安装应用
1. 用数据线连接 iPhone 和 Mac
2. 打开 Sideloadly
3. 点击「Browse」选择 `EduMindIOS.ipa` 文件
4. 在「Device」下拉菜单选择你的 iPhone
5. 在「Apple ID」栏输入你的 Apple ID 和密码
6. 点击「Start」开始安装
7. 在 iPhone 上信任开发者证书

## 定期续签（每 7 天）

1. 打开 Sideloadly
2. 用数据线连接 iPhone 和电脑
3. 选择之前安装的应用
4. 点击「Refresh」或「Re-sign」按钮
5. 输入 Apple ID 密码
6. 等待续签完成

## 高级功能

### 自动续签（Sideloadly Pro）
- Sideloadly 提供付费版本，支持自动续签
- 价格约 $3-5/月
- 无需手动每 7 天操作

### 多设备安装
- 可以在同一台电脑上为多个设备安装
- 每个设备使用各自的 Apple ID

## 常见问题

### Q: Sideloadly 和 AltStore 有什么区别？
A:
- Sideloadly 支持 Windows 和 Mac
- AltStore 只支持 Mac
- Sideloadly 操作更简单（图形界面）
- AltStore 功能更强大（支持插件等）

### Q: 7 天后应用会怎么样？
A: 应用将无法打开，需要重新续签。续签后即可正常使用。

### Q: 可以永久使用吗？
A: 可以，但需要每 7 天续签一次。这是苹果免费开发者证书的限制。

### Q: 安全吗？
A: Sideloadly 是知名工具，广泛使用。但使用任何第三方工具都有风险，请自行评估。
